# warlock
A clone of Warcraft3 map "Warlock" with [CreateJS](https://www.createjs.com/) &amp; [Socket.IO](https://socket.io/)

## get started
1. install environment
```
npm install
```

2. build client
```
gulp build
```
or for development
```
gulp watch
```

3. start server
```
npm start
```

4. enjoy it
```
http://localhost/
```
